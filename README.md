# 🧠 AI Intelligence Agent

A fully automated AI-powered news intelligence system that generates daily market-aware reports and sends them directly to Telegram.

It collects global news, analyzes sentiment using AI, and converts it into structured intelligence reports.

---

# 🚀 Features

- 🌍 World News aggregation (Google News + TechCrunch)
- 💼 Business & Economic updates
- 🤖 AI & Technology tracking
- 🪙 Crypto & Market insights
- 📊 AI-powered sentiment analysis (Bullish / Bearish / Neutral)
- 📈 Global market sentiment score (1–10)
- 🔥 Top story extraction
- 📩 Automatic Telegram delivery
- ⏰ Scheduled daily execution via GitHub Actions

---

# 🧠 How It Works

1. Collects news from RSS feeds  
2. Extracts and cleans headlines  
3. Sends data to Google Gemini AI  
4. AI generates structured intelligence report  
5. Sends final report to Telegram automatically  

---

# ⚙️ Tech Stack

- Python 3.11  
- Google Gemini AI (`google-genai`)  
- Feedparser (RSS parsing)  
- Requests (API calls)  
- GitHub Actions (automation)  
- Telegram Bot API
